A good start

Our teacher Mr. Mueller has taught us many tricks and tips when we code. In this class it would be a big help if you knew some keyboard shortcuts to help you code for maximum effeciency. Here is a link to some keyboard shortcuts for mac users. These will help you complete tasks as fast as possible. Another good way to start clas is to have your computer tabs up and running so you wont fall behind. You should utilize the bookmarks bar. Bookmark the following: Google, your class website, c9.io (your terminal) and Github.com.

Class Lessons

In this class it is easy to fall behind so you should always be engaged in the class. Once you enter the room your instructions will be up on the board. Listen to your teacher carefully and learn with the class. It is always good to venture coding on your own but you must keep on track with the teachers lessons. Through the class we will have class discussions and time to practice coding what we learned. When learning new languages in code our teacher would make us practice on a website called CodeAcadamy. This is a great webstite to visit by yourself and learn as well. If you learn some languages that will give you a huge head start on coding. Some you should learn are: Java, Jquery, Python, Markdown Syntax, html, css. robotics.
